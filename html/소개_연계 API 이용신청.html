연계 API 이용 신청

통합관리포털에서 전자정부 연계 API 이용을 신청하는 방법을 안내합니다.

사전준비가 무엇인가요?

이용 절차 확인하기

아직 통합관리포털 회원가입을 안 했어요.

회원가입하기

민간 SaaS 연계공통기반 및 전자정부 연계 API 이용 사전준비를 마친 민간 SaaS 기업과 이용기관은 통합관리포털에서 전자정부 연계 API 이용 신청을 할 수 있습니다.

  • 전자정부 연계 API 이용 신청을 진행하기 위해서는 민간 SaaS 연계공통기반 통합관리포털에 회원가입 되어있어야 합니다.
  • 회원가입만 진행한 경우 일반회원 등급으로 전자정부 연계 API 신청이 불가합니다.
  • 이용기관 담당자의 보안성 검토 결과서 등록 이후, 이용기관 및 민간 SaaS 담당자는 정식회원으로 전환되며 민간 SaaS 담당자의 전자정부 연계 API 신청이 가능합니다.

전자정부 연계 SaaS 신청 및 이용은
다음과 같이 진행됩니다.
