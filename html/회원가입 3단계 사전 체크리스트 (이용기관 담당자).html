  • [필수] 1. 통합관리포털 회원가입 체크사항

    • 1-1. 행정·공공기관임을 검증할 수 있는 서류(공공기관 고유번호증/사업자등록증 등)을 구비하였는가?

  • [필수] 2. 공공부문 이용가능 SaaS 체크사항

    • 2-1. 이용 예정 기업의 민간 SaaS는 ‘국가공공기관 민간클라우드 컴퓨팅서비스 도입 가능 목록’에 등록되어 있는가?

  • [필수] 3. 보안성 검토 체크사항

    • 3-1. 민간 SaaS 사용을 위한 보안성 검토를 추진할 준비를 하고 있으며 관련 서류(사업계획서, RFP, 자체보안대책, 시스템 구성도 등)에 대해 파악하고 있는가?

  • [필수] 4. 전자정부 연계 API 이용신청 체크사항

    • 4-1.

      이용 예정인 전자정부 연계 API(LDAP, 모바일공무원증, 행정표준코드, GPKI 인증, EPKI 인증)를 선택하였는가? ※ 공공기관은 행정표준코드만 이용가능

    • 4-2.

      필요한 서류(이용계약서, 서비스수준협약서(SLA), 보안성 검토 결과서)를 준비하고 있는가? ※ 이용계약서 및 서비스수준협약서(SLA)는 기관 자체 서식으로 제출

    • 4-3. 전자정부 연계 API 이용신청에 필요한 이용계약서 및 서비스수준협약서(SLA)를 준비하고 있는가?
