통합관리포털 소개

민간 SaaS 연계공통기반을 운영하는
통합관리포털에 대해 소개합니다.

민간 SaaS 연계공통기반 통합관리포털은
민간 SaaS 연계공통기반에서 정부 공통시스템 정보(데이터)를 제공하는 전자정부 연계 SaaS의 정보를 제공하고,
민간 SaaS 연계 현황 관리, 연계 서비스 신청 및 이용, 계정 관리 등을 통합적으로 관리하기 위한 포털 입니다.

클라우드 정책협의체는 다음과 같은 기관과 역할로 이루어집니다.
총괄·운영기관
  • 행정안전부
  • 한국지능정보사회진흥원(NIA)
정보보유기관

정부디렉터리(LDAP), 행정표준코드, 모바일 공무원증 등
정부 공통시스템 서비스 제공 기관

이용기관

전자정부 연계 SaaS와 연계된 공공용 민간 SaaS를 이용하는
행정‧공공기관

민간 SaaS 기업

전자정부 연계 SaaS와 공공용 민간 SaaS간 연계 개발을 하고, 연계된 전자정부 연계 SaaS를 제공하는 기업

사용자별 포털 기능

이용현황 관리 기능

  • 정부 공통시스템과 연계된 민간 SaaS 관리
  • 민간 SaaS를 이용하는 이용기관 관리
  • 이용기관이 사용중인 민간 SaaS 정보 관리

운영·관리 기능

  • 포털 사용자 및 사용자 권한 관리
  • 통합관리포털 메뉴 관리
  • 시스템, 서비스 이용신청, API 운영 및 관리
  • 장애, 병목 대응을 위한 로그 관리

API 관리 기능

  • SaaS 서비스별 API 정보 관리
  • MS API 조합을 위한 MS API 경로 관리

API 처리 기능

  • 서비스 시작, 종료 등을 수행할 수 있는 관리
  • 장애 시 멈춤, 재실행 등 처리

쾌적한 민간 SaaS 연계공통기반 통합관리포털 이용을 돕기 위해 이용 가이드를 제공하고 있습니다.
